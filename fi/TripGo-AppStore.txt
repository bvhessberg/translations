# DESCRIPTION

Mikä tekee TripGon parhaaksi reittiopas appiksi?

TripGo on all-in-one liikenteen ja kartta ratkaisu, joka integroituu elämääsi ja antaa sinulle tiedot, joita tarvitset, kun sitä tarvitset.

Palkittu sovellus avulla voit matkustaa älykkäämpi liittämällä kalenterin reaaliaikaisen liikenteen ehdotuksia, jotka tarjoavat sinulle vaihtoehtoja helpon vertailun.

Kerro TripGo:lle teidän liikenteen mieltymykset; aikataulu; ajoneuvojen ja lempipaikoja ja se antaa sinulle ovelta ovelle reittiohjeet sopivaksi elämäntapa.

Toimii kaikkialla yhä enemmän suurissa kaupungeissa Australiassa, Kanadassa, Euroopassa, Uudessa-Seelannissa, ja Yhdysvalloissa.


LIIKENNE Tuetut tilat:

TripGo tukee seuraavia liikennemuotojen:

+ Joukkoliikenne: bussit, lautat, metro, junat, raitiovaunut ja enemmän
+ Taksi
+ Shuttles
+ Auto
+ Auton jakaminen
+ Moottoripyörä
+ Polkupyörien
+ Kaupunkipyöräjärjestelmä
+ Käveleminen

Saat tulokset kutakin näistä vaan myös niiden yhdistelmöistä.


OMINAISUUDET:

+ Automaattisia matkoja kodin/työpaikan/tapahtumien välillä kalenterissa
+ Ovelta ovelle vaihtoehtoja - helposti vertaile hinnat, aikaa ja ympäristövaikutukset - kuten julkinen liikenne, taksi ja kaupunkipyöräjärjestelmät.
+ Reaaliaikainen joukkoliikenteen tiedot
+ Joukkoliikenteen aikatauluista
+ Suunnitelmat matkoja mukaan teidän liikenteen mieltymykset.
+ Saat muistutuksia tulevista suunniteltuista matkoista, joten voit aina saapua tapahtumiin ajoissa.
+ Joukkoliikenne hinnoittelu
+ Tietullejen ja parkkipaikkojen tiedot
+ Taksit
+ Etsi kiinnostavia kohteita ja yrityksiä
+ Avaa Apple Mapsista
+ Käynnistä turn-by-turn navigoinnin
+ Universal iPhone ja iPad
+ Australiassa tehty
