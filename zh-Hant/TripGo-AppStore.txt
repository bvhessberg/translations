# DESCRIPTION

是什麼讓TripGo成為最佳的旅程規劃App？

TripGo是個整合您生活並可以隨叫隨到的完整大眾交通系統使用方案提供者

我們屢獲殊榮的App將即時旅程建議整合到您的行事曆中，讓您更智慧地往來各地及更方便的比較旅程方案．

讓TripGo知道您偏好的交通工具，日程表，擁有的車輛與經常到訪的地方，TripGo將會提供配合您生活方式的點對點旅程建議．

持續增加對以下國家主要城市的支援：澳大利亞，加拿大，歐洲，紐西蘭及美國．

支援的運輸模式:

TripGo支援以下的交通工具:

+ 大眾交通: 公車，渡輪，地鐵，火車，輕軌及更多
+ 計程車
+ 區間車
+ 汽車
+ 汽車共享
+ 機車
+ 腳踏車
+ 腳踏車共享
+ 步行

我們所建議的旅程並不侷限于單一交通工具，我們同時考量任何可由以上任何交通工具所組合而成的結果．

特點：

+ 支援即時大眾交通訊息
+ 支援搜尋地標及公司行號
+ 支援將旅程儲存在行事曆
+ 支援從Apple地圖中開此本App
+ 支援從App中開啟Apple地圖導航
+ 支援iPhone及iPad
+ 提供大眾交通票價資訊
+ 提供通行費及停車場資訊
+ 提供計程車車費
+  提供點到點的旅程選擇（包含大眾交通，計程車，腳踏車分享等等），讓您能更方便的比較價錢，旅程時間，以及對環境的影響．
+ 在旅程開始前通知您，所以您可以準時的到達您的目的地
+ 配合您所偏好運輸模式的旅程規劃員
+ 澳大利亞設計
