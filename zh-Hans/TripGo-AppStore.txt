# DESCRIPTION

是什么让TripGo成为最佳的旅程规划App？

TripGo可以整合您的生活，随时随地为您提供全方位的公共交通解决方案

我们屡获殊荣的App将即时旅程建议整合到您的日历中，让您更智慧地往来各地及更方便地比较不同旅程方案．

让TripGo知道您偏好的交通工具、日历、拥有的车辆以及经常去往的地方，TripGo将会提供配合您生活方式的点对点旅程建议．

不断增加对以下国家主要城市的覆盖：澳大利亚，加拿大，欧洲，新西兰及美国．

支持的运输模式:

TripGo支持以下的交通工具:

+ 公共交通: 公共汽车，轮渡，地铁，火车，轻轨及更多
+ 计程车
+ 区间车
+ 汽车
+ 汽车共享
+ 摩托车
+ 自行车
+ 自行车共享
+ 步行

我们所建议的旅程并不局限于单一交通工具，我们同时考虑由以上交通工具可形成的不同组合．

特点：

+ 支持即时公共交通信息
+ 支持搜寻地标及公司名称
+ 支持旅程储存到日历中
+ 支持从苹果地图中开启本App
+ 支持从App中开启苹果地图导航
+ 支持iPhone及iPad
+ 提供公共交通票价信息
+ 提供通行费及停车场信息
+ 提供计程车车费
+ 提供点到点的旅程选择（包含公共交通，计程车，自行车分享等等），让您能更方便的比较费用，旅程时间，以及对环境的影响．
+ 在旅程开始前给您提示，让您可以准时到达目的地
+ 配合您所偏好的交通工具的个性化出行管家
+ 澳大利亚设计
